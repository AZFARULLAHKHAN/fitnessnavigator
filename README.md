Fitness Navigator

Fitness Navigator is a basic web application that gives workout and diet suggestions based on user input. The goal is to make fitness guidance simple and easy to access for anyone.

What it does

Takes user input like fitness goals and preferences
Suggests workouts based on that input
Provides a simple diet plan
Gives personalized recommendations instead of general advice
Runs quickly and keeps things straightforward

Built with

Python
Flask
HTML and CSS
SQLite (if used for storing data)

How to run

Clone the repository
git clone https://github.com/AZFARULLAHKHAN/fitnessnavigator.git
Go to the project folder
cd fitnessnavigator
Install the required packages
pip install -r requirements.txt
Run the application
python app.py
Open your browser and go to
http://127.0.0.1:5000/
